# Winget Package

This directory contains the Winget manifest templates for salesforce-cli.

## Package ID

`OpenCLICollective.salesforce-cli`

## Installation

```powershell
winget install OpenCLICollective.salesforce-cli
```

## How Releases Work

Winget manifests are submitted to the [microsoft/winget-pkgs](https://github.com/microsoft/winget-pkgs) repository.

The release workflow uses `wingetcreate` to:
1. Update the version and URLs
2. Calculate SHA256 checksums
3. Submit a PR to winget-pkgs

The manifests in this directory are templates for reference. The actual submission is generated by `wingetcreate`.

## Manual Submission

If you need to submit manually:

```powershell
# Download wingetcreate
iwr https://aka.ms/wingetcreate/latest -OutFile wingetcreate.exe

# Update and submit
./wingetcreate.exe update OpenCLICollective.salesforce-cli `
  --version <VERSION> `
  --urls "https://github.com/open-cli-collective/salesforce-cli/releases/download/v<VERSION>/sfdc_<VERSION>_windows_amd64.zip|x64" `
         "https://github.com/open-cli-collective/salesforce-cli/releases/download/v<VERSION>/sfdc_<VERSION>_windows_arm64.zip|arm64" `
  --submit `
  --token YOUR_GITHUB_TOKEN
```

The token needs permission to create PRs in microsoft/winget-pkgs.
